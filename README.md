# Proyecto-1-Analisis-de-redes-sociales
Integrantes:Samir Nassar, Miguel Figueroa. Enlace del repositorio:
https://github.com/Miguelfa16/ProyectoGrafos.git
Descripción: Este es un programa de escritorio para gestionar y analizar una red social. La aplicación permite cargar una red de usuarios desde un archivo de texto, mostrarla visualmente y modificarla, permitiendo agregar o eliminar usuarios y sus relaciones.Todos los cambios realizados se pueden guardar de nuevo en el archivo. La principal herramienta de análisis de la aplicación es su capacidad para identificar "comunidades" (componentes fuertemente conectados) y mostrarlas automáticamente en el grafo usando diferentes colores.
