Integrantes:Samir Nassar, Miguel Figueroa. Enlace del repositorio: https://github.com/Miguelfa16/ProyectoGrafos.git

Programa en Java para gestionar y analizar una red social desde un archivo .txt, permitiendo modificar usuarios y relaciones. 
El sistema visualiza el grafo e identifica "comunidades" (componentes fuertemente conectados) mediante colores.
